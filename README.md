# Pterodactyl Installer ü¶ñ

Welcome to the **Pterodactyl Installer** repository! üéâ This installer is designed to help you easily install and set up the Pterodactyl Panel on **Debian-based** or **RHEL-based** machines. If you encounter any issues during the installation process, please refer to our troubleshooting section for helpful tips. üõ†Ô∏è

## Installation üìù

Follow these steps to get started:

1. üßπ **Prepare your machine**: Ensure that your machine is freshly reinstalled if you've made any changes to it beforehand. A clean setup will help avoid conflicts during installation.

2. üåê **DNS Configuration**: Point a **DNS A-Record** to your machine's IP address. For example, you can point `panel.example.com` to `0.0.0.0`.

3. üì• **Download and Run the Installer**: To download and run the installer, simply enter the following command into your terminal and follow the prompts:
```
bash <(curl -Ss https://raw.githubusercontent.com/sensei-this/pterodactyl-installer/main/install.sh || wget -O - https://raw.githubusercontent.com/sensei-this/pterodactyl-installer/main/install.sh) auto
```

# üìå Database Installation on Pterodactyl

## 1Ô∏è‚É£ Preparing the Server

üîπ Make sure you have **Pterodactyl Panel** and **Wings** installed.  
üîπ Install **MySQL** or **MariaDB** on the panel server.  
üîπ Check if port **3306** (or another if using a custom one) is open.  

---

## 2Ô∏è‚É£ Configuring Database Access

üîπ Open the database configuration file:  
```bash
sudo nano /etc/mysql/mariadb.conf.d/50-server.cnf
```

üîπ Find the following line:
```ini
bind-address = 127.0.0.1
```
If you have an IP replace `127.0.0.1` with your IP address (e.g., `Example --> 0.0.0.0`).  
If you have a domain replace `127.0.0.1` with your IP address (e.g., `Example --> 86.144.10.13`). 

üîπ Save the changes (Ctrl + X ‚Üí Y ‚Üí Enter).  

üîπ Restart MySQL/MariaDB:  
```bash
sudo systemctl restart mysql
```
```
sudo systemctl restart mariadb
```

üîπ Open MySQL:  
```bash
mysql -u root -p
```

üîπ Select the MySQL database:  
```sql
USE mysql;
```

üîπ Create a new user:  
```sql
CREATE USER 'username'@'%' IDENTIFIED BY 'password';
```

üîπ Grant all privileges:  
```sql
GRANT ALL PRIVILEGES ON *.* TO 'username'@'%' WITH GRANT OPTION;
```

üîπ Flush privileges:  
```sql
FLUSH PRIVILEGES;
```

üîπ Exit MySQL:  
```sql
EXIT;
```

---

## 3Ô∏è‚É£ Creating a Database in the Panel

1. Log into Pterodactyl Panel as an **administrator**.  
2. Go to **Admin Panel** ‚Üí **Databases**.  
3. Click **Create Database**.  
4. Specify:  
   - **Database Name**  
   - **Server to link the database to**  
   - **Database Host** (your IP or `127.0.0.1` if local)  
5. Save the changes.  

---

## ‚úÖ Done!

Your database is now set up and ready to use! üöÄ

## Troubleshooting ‚ö†Ô∏è

1) If you receive a **MySQL connection error** üíª, you may have run the installer multiple times. In this case, the best solution is to **reinstall your operating system** and run the install script again. üîÑ

2) If you encounter **Let's Encrypt SSL generation errors** üîí, it could be because you're using an **IP address** instead of a domain, or because the **FQDN** you're trying to generate an SSL for doesn't have an **A-Record** pointing to your machine IP. üåç
